#Arduino-Controller

###Arduino controller is an Android Application that can send data from android device to your arduino . It also Includes a feature of 6 Voice Commands.

####TURN LEFT
####TURN RIGHT
####MOVE FORWARD
####MOVE BACKWARD
####ROTATE RIGHT
####ROTATE LEFT


####Table:


| Button        | DataToArduino | 
| ------------- |:-------------:| 
| UP            |    'U'        |
| DOWN          |    'D'        | 
| LEFT          |    'L'        |
| RIGHT         |    'R'        |
| RotateRight   |    'i'        |
| RotateLeft    |    'e'        |
